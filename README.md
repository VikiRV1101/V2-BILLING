# Offline Bakery Shop Management System

A complete, 100% offline POS and management system for bakery shops.

## How to Run Offline

1. **Extract all files** to a folder on your computer.
2. **Open `login.html`** using any modern web browser (Chrome, Edge, Firefox).
3. **No Internet Required**: All data is stored locally in your browser's IndexedDB.

## Features Included
- **POS Billing**: Modern interface with cart, search, and stock integration.
- **Product Management**: Support for unit types, pack sizes, and images.
- **Category Management**: Easy organization.
- **Customer Orders**: Track advance payments and delivery status.
- **Staff Management**: Add staff and track their attendance.
- **Expense Tracking**: Record daily costs.
- **Reports**: Generate sales, stock, and profit reports with CSV export.
- **Thermal Printing**: Support for 58mm/80mm thermal receipts and A5 invoices.
- **Backup & Restore**: Mandatory safety system to export/import all data as JSON.

## Safety Note
> [!CAUTION]
> Since data is stored in the browser (IndexedDB), clearing your browser's site data or "cookies" may delete the application data. **Always use the Backup feature daily** to download a JSON copy of your data for safety.
